## About

This repository contains automated BDD test suites for the SauceDemo web shop (https://www.saucedemo.com/), comparing three frameworks: Cucumber, Enhanced Gherkin, and ConcordiaLang. The work was completed as part of a research project at SPbPU Institute of Computer Science and Cybersecurity (SPbPU ICSC).
